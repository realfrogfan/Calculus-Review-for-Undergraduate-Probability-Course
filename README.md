# Calculus Review for an Undergraduate Probability Course
Welcome! This repository contains comprehensive calculus review materials specifically designed to prepare you for an undergraduate probability course. 

The concepts here build sequentially—starting from fundamental univariate calculus and advancing to the bivariate techniques (like double integrals and Jacobians) that are essential for continuous probability distributions.

### How to Use These Materials
Simply click any of the links above to open the corresponding review section. The pages include instructional text, mathematical formulas, and rendered visualizations. For the activities, the accompanying R code is provided so you can see exactly how the visual models are generated.
